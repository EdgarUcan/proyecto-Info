Aplicaciones Inteligentes

Las aplicaciones inteligentes están muy relacionadas con el internet de las cosas, y vienen con el objetivo de mejorar cada vez más nuestra calidad de vida y hacer nuestro trabajo más fácil. Por mencionar algunos ejemplos de las Smart Apps en nuestra vida cotidiana tenemos: Apps para el monitoreo y registro de nuestra salud personal a través de diferentes accesorios que se pueden vincular al teléfono móvil, aplicaciones con las que podemos automatizar tareas como encender las luces de nuestra casa cuando el GPS detecte que hemos llegado ¿no es increíble?. También nos encontramos con algunas aplicaciones de IoT (Internet de las cosas) como: Refrigeradores domésticos que detectan su contenido y, de acuerdo a nuestros gustos, se encargan de manera autónoma de las compras por internet, también encontramos semáforos inteligentes que detectan percances vehiculares y que pueden redistribuir el tráfico hacia rutas alternas, mediante notificaciones a una App de ruta.

Tipos de Aplicaciones móviles

Actualmente existen tres tipos de app:

App Nativas

Son aquellas aplicaciones que han sido desarrolladas para un sistema operativo en específico (iOs, Android, Windows); para que una app esté disponible en todos los sistemas operativos, se debe crear una app para cada sistema operativo. Este tipo de aplicaciones solo se pueden encontrar en tiendas de app, como App Store o Google Play.

La mayor ventaja que tiene este tipo de apps es que el usuario puede acceder al hardware del móvil como agenda, GPS, cámara entre otros. Además algunas de estas apps no necesitan conexión a internet a la hora de usarlas.

App Web

También es conocida como web apps, este tipo de aplicación se ejecuta a través de un navegador (Safari, Google Chrome, Opera, Firefox); por ello puedes acceder a cualquier aplicación web sin importar el sistema operativo que uses. Solo necesitas acceder a tu navegador favorito mediante una URL.

Desarrollar una app de este tipo es muy económico, pero no funciona si no se está conectado a internet. Puedes usar estas apps en ordenadores, tablet y smartphone que tengan acceso aún navegador de internet.

App Híbridas

Este tipo de aplicación es una combinación de las dos anteriores; entonces, podemos decir que son aplicaciones desarrolladas con un lenguaje de programación Javascript, CSS o HTML.

Además son aplicaciones multiplataforma porque se adaptan a cualquier sistema operativo, por lo tanto, pueden ser utilizadas en la PC y en dispositivos móviles porque las puedes usar desde el navegador o descargarlas en una tiende de app. Un ejemplo, de aplicaciones híbridas es Facebook e  Instagram que se puede usar en el pc

Otros tipos de aplicaciones móviles que debes conocer

Aunque solo existen tres tipos básicos de aplicaciones, elegir qué tipo de aplicación móvil necesitamos según las necesidades, si no que también podemos optar por construir o desarrollar alguna aplicación a conveniencia y esas están en otra categoría de aplicaciones.

Según sus autores

Existen personas o incluso empresas que se trabajan en el desarrollo de aplicaciones para después venderlas a otros. Estas aplicaciones están diseñadas para facilitar tareas complejas y hacer mas sencilla la experiencia de los usuarios.

Educativas

Es un programa multimedia, pensado para ser usado a través de dispositivos electrónicos y también como una herramienta de e-learning, con la creciente popularidad del uso de la tecnología móvil, ha posibilitado la aparición de un nuevo enfoque educativo conocido como m-learning. Los cuales sun fin es de aprender y mejorar en el ambito del estudi y no solo lo usen para otras cosas, sino que aprovechen al maximo todo lo que tienen

 

De ocio y entretenimiento

Son programas que tenemos al alcance a través de nuestros dispositivos y que nos ayudan a pasar tiempo y que representan para nosotros diversión, sana distracción, disfrute e incluso representan un magnifico apoyo para sobrellevar las cargas de la vida diaria.

Edición de fotos y video

Un editor es un programa que permite crear o realizar modificaciones y mejoras a nuestros archivos.

Localización

Este tipo de aplicaciones nos ofrecen la localización de nuestro o cualquier dispositivo mientras este encendido. Para ello se requiere el uso del GPS para ubicar en el mapa el dispositivo a localizar, tienen buena exactitud dependiendo de la cobertura donde este el dispositivo.
